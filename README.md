# 🧩 Jogo de Labirinto em Java

## 📌 Descrição

Este é um jogo de labirinto desenvolvido em Java para rodar no terminal.
O jogador deve encontrar a saída do labirinto antes que seus movimentos acabem.

---

## 🎮 Como jogar

* `@` → Jogador
* `S` → Saída
* `#` → Parede
* `.` → Caminho
* `*` → Item

### Controles:

* W → Cima
* A → Esquerda
* S → Baixo
* D → Direita
